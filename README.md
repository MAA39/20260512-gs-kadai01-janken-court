# 逆転じゃんけん — 説得の心理戦

gsアカデミー課題01: jQuery（JavaScript）課題

## 概要

99連勝の絶対王者「闇のジャンケン王 黒崎 玄武」を、言葉で「負けたい」と思わせるゲーム。
4軸の心理パラメータ（闘志/警戒/共感/譲歩）を Structured な選択肢効果で動かし、説得 → じゃんけん本番の2フェーズで決着する。

## 遊び方

1. 難易度（3/5/10ターン）を選ぶ
2. 選択肢で黒崎の心理を崩していく
3. 譲歩 ≥ 70 かつ 共感 ≥ 50 で説得成功
4. 警戒 ≥ 95 で説得失敗
5. 最後にじゃんけんで決着

## 技術

- 素のHTML/CSS/JavaScript（1ファイル完結）
- 外部依存ゼロ（Google Fontsのみ）

## デプロイ

Vercelで静的配信。
